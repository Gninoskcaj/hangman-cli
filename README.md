# Installation

```bash
$ git clone https://github.com/Gninoskcaj/hangman-cli
$ chmod +x ./hangman-cli/hangman
$ mkdir ~/bin
$ sudo mv ./hangman-cli/ ~/bin/
$ 'export PATH=$PATH":$HOME/bin/hangman-cli"' >> .profile
$ source .profile
```

Thats it for installation. To start playing type `hangman`
